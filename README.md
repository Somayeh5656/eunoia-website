# Eunoia – Calming Emotional Support Website

This is a React + Vite frontend for the Eunoia project – a warm, empathetic AI companion for emotional well-being. It includes a custom ElevenLabs conversational widget, resource links, and crisis support information.

## Getting Started

### Prerequisites

- Node.js 18 or higher
- npm or yarn

### Installation

Clone the repository

Install dependencies
npm install

Start the development server:
npm run dev
Open your browser and go to http://localhost:5173.
